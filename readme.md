# How to Install

Run this command:

```
pip install -r requirements.txt
```
Download [model](https://github.com/ultralytics/assets/releases/download/v8.2.0/yolov8n-cls.pt)

## How to run:

Run this command:

```
flask run
```
## How to run Docketfile :

Run this command:

```
docker build -t  WebsitFaceAnalyez
```
Run this command:

```
docker run -p 8080:5000
```
